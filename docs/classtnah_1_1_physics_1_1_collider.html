◆ m_LocalPosition

tnah::Physics::Collider::m_LocalPosition = {}
private

The position of the Collider locally to its parent Rigidbody.

Note
The position should always be at 0,0,0 ie the centre of the Rigidbody.
However the position should be changed if the object requires multiple colliders with different positions relative to the Rigidbody.
